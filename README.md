## 🎯 Overview

This is a **demo website** designed to demonstrate how a modern plumbing service website should look, feel, and convert visitors into leads.

Built with **HTML5, Tailwind CSS, and vanilla JavaScript** — no frameworks, no build tools, no complexity. Just clean, professional code that any plumber can see and say: *"I want this for my business."*

---

## ✨ Features

### Design & UX
- **Responsive design** — Mobile-first approach, works flawlessly on all devices
- **Glassmorphism navbar** — Transparent at top, blurred glass on scroll
- **Split layout sections** — Title left, content right (consistent throughout)
- **Bento grid services** — Modern, asymmetric card layout
- **Dual marquee testimonials** — Auto-scrolling reviews, pauses on hover
- **Timeline how-it-works** — Sticky left content, scrolling right timeline
- **Interactive service area map** — Real Google Maps embed with hover effects
- **Scroll reveal animations** — Sections fade and slide in as you scroll
- **Active nav highlight** — Current section automatically highlights in navbar
- **Smooth scroll** — All navigation links scroll smoothly to sections

### Lead Capture
- **Booking form** — Full lead capture form with validation
- **Form success state** — Animated success message with no backend needed
- **Newsletter signup** — Email capture in footer
- **Floating WhatsApp button** — Always visible, pre-filled message
- **Emergency CTA cards** — Multiple call-to-action points throughout
- **Scroll-to-top button** — Easy navigation back to top

### Technical
- **Single-page architecture** — Fast loading, simple deployment
- **No dependencies** — Pure HTML, CSS, JavaScript
- **No build tools required** — Works directly in browser
- **Optimized images** — WebP format, lazy loading ready
- **SEO meta tags** — Open Graph tags for social sharing
- **Accessible** — Semantic HTML, ARIA labels, keyboard navigation
